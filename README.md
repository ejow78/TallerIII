RepairIT

Sistema de Gestión y Trazabilidad para Talleres de Servicio Técnico de Hardware
Descripción

RepairIT es una plataforma web integral diseñada para la automatización del ciclo de vida de reparaciones de hardware. El sistema permite registrar equipos, gestionar estados de reparación mediante un flujo de trabajo dinámico, y mantener un historial técnico trazable.
Facilita la comunicación entre el técnico y el cliente mediante enlaces de seguimiento únicos, y optimiza la gestión del taller a través de un control de inventario de insumos y componentes vinculado directamente a cada reparación.

Stack Tecnológico

Este proyecto utiliza el stack MERN para garantizar escalabilidad y flexibilidad:

    Frontend: React.js con Tailwind CSS

    Backend: Node.js

Características Principales

    Ingreso Unificado: Formulario optimizado para carga de especificaciones técnicas (CPU, GPU, RAM, etc.) y datos del cliente.

    Seguimiento en Tiempo Real: Generación de URLs únicas por reparación para consulta pública del cliente.

    Control de Inventario: Descuento automático de insumos (pasta térmica, pads, etc.) por reparación.

    Panel Administrativo: Dashboard para gestión de estados (Diagnóstico, Presupuesto, Listo, etc.).

Instalación

    Clonar el repositorio:
    git clone https://github.com/tu-usuario/trazati.git

    Instalar dependencias del frontend:
    cd frontend
    npm install
